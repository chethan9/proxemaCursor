# WooSync - Multi-Tenant WooCommerce Integration Platform

## Vision
Operations console for agencies managing multiple WooCommerce stores. Sync commerce data to centralized Postgres, monitor sync health, troubleshoot issues.

**Users:** Agency teams, eCommerce operators, multi-store managers

## Features
- Client → Site hierarchy management
- Site onboarding (WooCommerce API keys)
- Multi-aspect sync engine (products, orders, customers, etc.)
- Sync run logs with status tracking
- Webhook registration and event logging
- Per-site data explorer
