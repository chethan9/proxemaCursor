# Sync, webhooks & developer tools

Visibility of these areas depends on permissions (`Sync`, `Webhooks`, `API`, etc.).

## Sync Runs

**Sync Runs** lists synchronization executions between Proxima Cursor and your WooCommerce-backed sources—helpful when diagnosing delays or partial updates.

**When things look wrong**

- Confirm the latest run finished successfully.
- Retry policies are configured by administrators—escalate repeating failures with timestamps.

## Webhooks & Activity

**Webhooks** configures inbound/outbound automation endpoints your organization relies on. **Activity** under Webhooks helps audit deliveries when troubleshooting integrations.

## API management

**API** exposes credentials and usage surfaces for programmatic access—intended for technical stakeholders even though this Help Center targets general users. Coordinate with developers before rotating keys.
