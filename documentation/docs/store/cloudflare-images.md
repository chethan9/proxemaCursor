# Cloudflare Images

Some deployments accelerate product imagery through **Cloudflare Images** mirroring. When enabled, administrators configure credentials and variants; store-facing pages may show mirrored URLs for faster loads.

**What end users should know**

- If previews look wrong after edits, wait for mirror propagation or ask an admin to trigger repair workflows where available.
- This section may not apply if your organization keeps images only in WooCommerce media.
